
# Swipe 
generate invoice quiclky with Swipe



## Local Installation


Clone
```bash
git clone https://github.com/adwait12345/swipe.git
```

Install Swipe with npm

```bash
cd/swipe

npm install

npm start
```

## Features

- generate quick invoices
- grouping in invoice
- easy product management
- friendly UI/UX
